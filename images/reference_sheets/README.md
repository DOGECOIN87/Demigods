# Reference Sheets

These images document the concept-art and trait-sheet development process. They are references only—not final interchangeable traits.

Final production assets must:

- contain no character names
- use the locked 1254×1254 master rig
- preserve consistent front-facing alignment and upper-left lighting
- be isolated by category on true transparent canvases
- remain fully interchangeable across the collection

Compressed WebP previews are stored here for practical repository browsing. The full-resolution source package is maintained separately during production.
