# Universal Avoid Block

Append this block to every asset-generation prompt.

```text
AVOID:
contact sheet, trait sheet, infographic, multiple assets, multiple variations, multiple characters, labels, captions, typography, frames, display cards, white background, gray background, rendered checkerboard, fake transparency, merged traits, baked-in face, baked-in hair, baked-in clothing, background scenery, character names, franchise names, side view, three-quarter view, action pose, tilted pose, inconsistent scale, perspective distortion, cropped edges, incomplete asset, blurry edges, pixelation, ground shadow, inconsistent lighting, front-right key light, anatomical nudity, explicit anatomy, extra fingers, missing fingers, duplicate limbs, malformed hands, or asymmetrical placement unless explicitly specified.
```
